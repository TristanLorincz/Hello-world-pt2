HelloWorld

History

Test programs have always existed in software development, but the traditoin of using "Hello, World!" as a test message was influenced by and example program in the 1974 book the C Programming Language. The example was taken from a 1974 Bell Laboratories internal memorandum by Brian Kernighan, one of the co-creators of the Unixoperating system.

Example

Here is a JavaScript example:

<!DOCTYPE HTML> <html> <body> <script> alert('Hello, World!'); </script> </body> </html>
